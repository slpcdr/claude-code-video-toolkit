## Technical Notes

### GIF to Video Conversion
```bash
# Convert GIF to MP4 for Remotion
ffmpeg -i claude-code-install.gif -movflags faststart -pix_fmt yuv420p -vf "scale=trunc(iw/2)*2:trunc(ih/2)*2" claude-code-install.mp4
```

### Asciinema to Video Options
1. **asciinema-player** → screen record the playback
2. **svg-term-cli** → SVG → video pipeline
3. **termtosvg** → Alternative format
4. **Re-record with script** → Most control

### Remotion Setup
```bash
cd remotion
npx create-video@latest  # If starting fresh
npm install
npm run studio  # Preview
npm run render  # Final output
```
